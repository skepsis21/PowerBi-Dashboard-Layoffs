Casino Operational Performance Dashboard

An end-to-end data analytics project transforming raw casino activity data into actionable operational insights using SQL and Power BI.
Business Scenario

Casino operators need real-time visibility into floor revenue, VIP player risk, and gaming performance. This project automates the extraction and visualization of key performance indicators (KPIs) to support data-driven decision-making on the casino floor.
Solutions Implemented

    SQL Automation: Developed SQL queries to aggregate daily revenue and identify high-risk VIP segments.

    Data Pipeline: Built a structured pipeline to clean and prepare data for visualization.

    Interactive Dashboard: Designed a Power BI dashboard to track floor performance, unit revenue, and VIP risk allocation.

Tools Used

    Database: SQLite / SQL

    Visualization: Power BI

    Version Control: GitHub

Technical Skills

    SQL: Proficient in complex aggregations, data cleaning, and relational querying.

    Data Modeling: Experienced in designing star schemas and efficient data relationships for BI.

    Business Intelligence: Skilled in translating raw data logs into stakeholder-ready performance KPIs.

Key Insights

    Floor Revenue: Performance metrics broken down by game type.

    VIP Risk: Automated identification of at-risk loyalty tiers.

    Operational Efficiency: Analysis of revenue per unit to optimize floor layout.

How to View

    Download the repository and open casino_report.pbix in Power BI Desktop.

    If the data does not load, go to Transform Data > Data source settings.

    Select casino.db and click Change Source to point it to your local file path.

    Click Refresh to populate the dashboard.
