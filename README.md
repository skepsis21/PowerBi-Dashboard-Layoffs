# HR Layoffs Power BI Dashboard

## Overview
This project analyzes global company layoffs using SQL-cleaned data and Power BI dashboards.

## Tools Used
- MySQL (Data Cleaning)
- Power BI (Visualization)
- GitHub (Version Control)

## Steps
1. Data cleaning in SQL (deduplication, standardization, null handling)
2. Data imported into Power BI
3. DAX measures created for KPIs
4. Interactive dashboard built

## Key Insights
- Layoff trends over time
- Industry impact
- Country-level layoffs
- Top affected companies
